# internet-radio-raspberry-pi

Internet radio using Raspberry Pi3 micro computer

Hardware prerequisites
* Raspberry Pi3
* PiFace CAD 2

Software prerequisites
* Python3
* mocp
* GPIO https://pypi.org/project/PiIR/
